# 01 객체와 객체 지향 프로그래밍
---
## 절차 지향 프로그래밍
* 일련의 과정을 시간의 흐름에 따라 프로그래밍하는 것
* 대표적으로 C언어가 있다.
## 객체 지향 프로그래밍
* '아침에 일어나서 학교를 가는 과정'을 프로그래밍한다고 했을 때 여러 객체들이 존재한다.
* 학생, 버스, 학교, 밥
* 객체들 사이에 관계가 발생한다.
* 학생이 버스를 타는 행위는 학생이 버스를 타게 되면 돈을 지불하고 버스는 학생이 타면 승객이 늘어나고 운수회사에 이익이 올라가는 등의 관계가 형성된다.
* 이러한 관계를 가지고 프로그래밍하는 것을 객체 지향 프로그래밍이라고 한다.
* C++, C#, python, javascript
## 객체 지향 프로그래밍 방법
* 객체에 대한 정의
* 객체들이 가져야 하는 속성, 특성들을 정의한다.
* 객체가 해야하는 일들을 구현한다.
* 객체간에 협력이 발생하므로 협력이 되는 부분을 파라미터가 넘어가거나, 객체가 넘어가던가 하는 것을 통해서 협력을 구현할 수가 있다.
